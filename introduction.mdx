## What is CEMAC?

CEMAC (Sistema de Gestión de Inventario y Análisis) is a comprehensive SaaS platform designed specifically for stationery and invitations businesses. Built with modern web technologies, CEMAC helps you manage inventory, track sales, monitor stock levels, and gain valuable insights into your business performance.

## Built for stationery businesses

CEMAC is specifically designed for businesses in the stationery, invitations, and paper goods industry. Whether you're managing a retail store or wholesale operation, CEMAC provides the tools you need to:

- **Organize your inventory** by categories, brands, and suppliers
- **Track sales** with detailed transaction history and customer records
- **Monitor stock levels** with automated low-stock alerts
- **Analyze trends** to make data-driven business decisions
- **Manage your team** with role-based permissions for administrators and employees

## Technology stack

CEMAC is built with cutting-edge web technologies to ensure reliability, security, and performance:

- **Node.js & Express** - Fast, scalable backend server
- **Firebase Authentication** - Secure user authentication and session management
- **Cloud Firestore** - Real-time NoSQL database for instant data synchronization
- **Modern JavaScript** - ES6+ modules for maintainable code
- **Responsive design** - Works seamlessly on desktop and mobile devices

<Note>
  CEMAC follows a SaaS architecture with a centralized API deployed on Vercel, ensuring high availability and automatic scaling.
</Note>
